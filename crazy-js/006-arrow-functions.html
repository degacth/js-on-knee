Стрелочные функции

  • Жёсткая привязка к this
  • Отсутствие переменной arguments
  • Если это функция одного аргумента, то круглые скобки можно опустить
    const sayHello = name => 'Hello ' + name
  • Две формы записи
    • Короткая форма
      const mult = (a, b) => a * b
    • Полная форма
      const div = (a, b) => { return a / b }
