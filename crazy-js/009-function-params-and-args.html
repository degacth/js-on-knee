Параметры и аргументы функций

  • Терминология
    • Параметры функции - это те переменные которые находятся в круглых скобках
      const someFunction = (param1, param2) => param1 / 2 - param2
    • Аргументы функции - это те значения с которыми вызывают функцию
      someFunction(10, 4)
    • Функции вызываются с аргументами, которые записываются в параметры функции
  • Параметры функций могут иметь значения по умолчанию
    const parseInt = (val, radix = 10) => [native code]
  • Параметры массивы имеют особый синтексис:
    const head = ([first]) => first
    const tail = ([first, ...remains]) => ramains
  • Параметры объекты также имеют особый синтаксис:
    const configure = (el, {width, style = 'black', fontSize = 24, height = fontSize + 10}) => ... =
  • Вариативная функции
    const max = (...vals) => vals.sort()[vals.length - 1]
    const helloForAll = (name, ...others) => 'Hello, ' + name + ' and ' + others.join() + '!'
